# 🐍 我的贪吃蛇游戏

一个功能完整、视觉精美的网页版贪吃蛇游戏，具有3D立体效果和完整的音效系统。

## ✨ 游戏特色

- 🌈 **七彩渐变蛇身** - 红、橙、黄、绿、青、蓝、紫循环变化
- 💎 **3D立体效果** - 圆角设计、径向渐变、高光阴影
- 👀 **动态蛇头** - 根据移动方向调整眼睛位置
- 🍎 **精美苹果食物** - 3D渐变效果，带叶子装饰
- 🎵 **背景音乐** - 轻快的循环旋律（可开关）
- 🔊 **完整音效** - 吃食物、奖励、游戏结束音效
- 🎁 **奖励机制** - 每达到100分速度提升10%
- 💾 **本地存储** - 自动保存最高分记录

## 🎮 游戏玩法

- 使用方向键 **↑ ↓ ← →** 控制蛇的移动
- 吃到食物得分（每个+10分）
- 撞墙或撞到自己游戏结束
- 每达到100分倍数，速度会提升，难度增加

## 🎯 游戏控制

- **开始游戏** - 开始新游戏
- **暂停** - 暂停/继续游戏（也可按空格键）
- **重新开始** - 重置游戏
- **🔊 音乐** - 开启/关闭背景音乐
